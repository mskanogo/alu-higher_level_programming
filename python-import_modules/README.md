# Python - import & modules

## Description
This project is part of the ALU Higher Level Programming curriculum.  
It introduces the concept of **modules** and how to **import functions, variables, and scripts** in Python.  
You will also learn how to use **command line arguments** and prevent code from executing when a file is imported.

## Learning Objectives
By the end of this project, you should be able to:
- Explain why Python programming is awesome  
- Import functions from another file  
- Use imported functions and variables  
- Create and use your own modules  
- Use the built-in function `dir()`  
- Prevent code from being executed when imported using `if __name__ == "__main__":`  
- Handle command line arguments with `sys.argv`
